## 📌 Overview

This project is an intelligent **ATS Resume Scanner & AI Agent** built using **n8n**, designed to automate resume screening and provide actionable feedback for candidates.

It helps:

* Recruiters → automate candidate screening
* Students/Freshers → evaluate and improve resumes

---

## 🔍 Features

* 📄 **Resume Parsing**

  * Extracts data from uploaded PDF resumes
  * Converts unstructured data into structured format

* 🧠 **AI-Based Resume Scoring**

  * Evaluates resumes based on:

    * Skills match
    * Keyword relevance
    * Role alignment
    * Formatting quality

* 📊 **Automated Insights**

  * Generates resume score
  * Identifies strengths & weaknesses

* 📧 **Email Notification**

  * Sends feedback to candidates with:

    * Score
    * Improvement suggestions
    * Next steps

* 🎯 **Student-Friendly Tool**

  * Helps freshers check resume quality before applying
  * Provides actionable recommendations

---

## ⚙️ Tech Stack

* **n8n** – Workflow Automation
* **Google Gemini / LLM** – AI Evaluation
* **JavaScript (Code Node)** – Data transformation
* **Email Integration (Gmail)** – Notifications
* **PDF Parser** – Resume extraction

---

## 🔄 Workflow Architecture

1. Form Submission (Resume Upload)
2. Extract Data from PDF
3. Data Cleaning & Field Mapping
4. AI Evaluation (LLM)
5. Resume Scoring Logic
6. Email Feedback Delivery
---

## 🚀 How to Use

1. Import the workflow JSON into n8n
2. Configure:

   * Email credentials
   * LLM API (Gemini/OpenAI)
3. Run the workflow
4. Upload a resume via form
5. Get automated analysis + email feedback
